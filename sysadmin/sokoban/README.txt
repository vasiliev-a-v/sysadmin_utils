Сокобан - консольная игра
Смайлик передвигает шарики. Чтобы выиграть уровень, необходимо задвинуть все камни в дом.
Чтобы запустить игру, необходимо запустить файл main.sh

main.sh - главный файл. Он отображает меню и подключает другие файлы программы.
menu.sh - файл главного меню программы
game.sh - движок самой игры
header.sh - в этом файле объявлены переменные
